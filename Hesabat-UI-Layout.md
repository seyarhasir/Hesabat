# حسابات — Hesabat
## UI Layout & Screen Specifications

> **Design Principles:**
> - RTL first (Dari/Pashto) — all layouts mirror for right-to-left
> - One-hand friendly — primary actions reachable with thumb
> - Large touch targets — minimum 48×48dp for all buttons
> - Works on small screens — minimum 5" screen, 360dp wide
> - High contrast — readable in bright Afghan sunlight
> - Minimal text — icons + numbers do most of the work

Variants:
- 🔄 Blue: `در حال همگام‌سازی...` — syncing
- ✅ Green (3 sec then hides): `همگام‌سازی شد` — done
- ⚠️ Orange (persistent): `آفلاین — X رکورد در انتظار` — offline
- 🔴 Red (persistent): `خطا — لمس کنید` — sync error, tap to retry
